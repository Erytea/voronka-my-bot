# Фитнес-Навигатор

Telegram-бот, который помогает принимать адекватные ежедневные решения для тела с учётом реального состояния, времени и ресурса.

## Описание

Фитнес-Навигатор — это спокойный, понимающий бот, который не мотивирует и не давит. Он помогает разобраться в реальном состоянии тела и предложить подходящее действие на день (тренировка, восстановление или осознанный отдых).

## Технологический стек

- Python 3.11+
- aiogram 3.x (асинхронный фреймворк для Telegram)
- Tortoise ORM (асинхронная ORM)
- SQLite (база данных)
- Aerich (миграции)

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository_url>
cd voronka-my-fitness-bot
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

5. Заполните `.env` файл:
```
BOT_TOKEN=your_telegram_bot_token_here
DATABASE_URL=sqlite://./db.sqlite3
```

6. Инициализируйте базу данных:
```bash
# Инициализация aerich (первый раз)
aerich init-db

# Для создания новых миграций в будущем:
# aerich migrate
# aerich upgrade
```

7. Запустите бота:
```bash
python -m src.fitness_navigator.main
```

Или из корня проекта:
```bash
cd src
python -m fitness_navigator.main
```

## Команды бота

- `/start` — приветствие и онбординг (сбор целей, ограничений, графика работы)
- `/checkin` — ежедневный чек-ин (состояние тела, доступное время, настрой)
- `/stats` — статистика и инсайты за последние 7/30 дней
